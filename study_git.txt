（创建文件夹）
mkdir 文件名

（显示当前目录）
pwd

（将某个目录变为git仓库）
git init

（将文件添加到仓库）
git add xxx.txt

（将文件提交到仓库）
git commit -m "提交说明"

（仓库内文件的状态）
git status

（仓库内文件发生的具体变化）
git diff

(显示从近到远的提交（commit）日志)
git log


（版本变换,HEAD表示当前版本号，上一个版本号是HEAD^,上上一个是HEAD^^,上100个是HEAD~100，）
git reset --hard 版本号


(显示从近到远的命令日志，注意开头是版本号)
git reflog

（某文件在工作区和版本库里最新版本的差别）
git diff HEAD -- xxx.txt


(把工作区的修改撤销)
git checkout -- xxx.txt

（把暂存区的修改撤销）
git reset HEAD file
